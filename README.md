# ☀️ LinMo Blog

  一个正在开发的React+TypeScript+Vite+SpringBoot博客

## ✨ 功能特性

- Login+后台页面UI
- 登录逻辑+token持久化
- Axios封装

## 👌 技术栈

### 前端
- React + TypeScript
- React Router
- React Reducer
- Axios

### 后端
- json-servre模拟接口（暂时）
